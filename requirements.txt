# 본 환경설정은 https://python.langchain.com/docs/get_started/quickstart에 설명이 되어 있다.

1. dotenv 설정
.env에는 OPENAI_API_KEY 를 설정한다.
.env를 사용하려면 pip install python-dotenv 를 설치해야한다.

from dotenv import load_dotenv
load_dotenv()


2. langchain 설치
pip install langchain

3. openai 설치
pip install openai

pip install streamlit

프론트엔드(Frontend)
사용자가 직접 보는 화면/인터페이스 쪽입니다.
예: 웹 브라우저에서 보이는 페이지(HTML/CSS/JS), 버튼, 입력창, 차트, React/Vue 같은 것들.
사용자의 클릭/입력을 받아서 백엔드에 요청을 보내고, 받은 결과를 예쁘게 보여주는 역할.
백엔드(Backend)
서버 쪽 로직과 데이터 처리를 담당합니다.
예: Python(Django/FastAPI/Flask), Node.js, DB 연동, 인증, 권한, 비즈니스 로직, 외부 API 호출(OpenAI, LangChain 등).
프론트엔드나 다른 클라이언트(모바일 앱, 다른 서버)가 HTTP 요청(API 호출)을 보내면, 그걸 받아서 처리하고 결과(JSON 등)를 돌려줍니다.
